
## 游戏引擎技术

### 主流引擎类型
- Unity：多用于移动 / 中小型游戏，优势是轻量化、跨平台适配便捷，版本迭代快（如 Unity 2022.x/2023.x）
- Unreal：多用于主机 / PC 大型游戏，优势是图形渲染能力强、物理引擎成熟，版本（如 Unreal 5.0/5.1）
- 自研引擎：大型厂商专属（如腾讯 QuickSilverX、网易 Messiah），适配特定游戏需求，定制化程度高

### 测试重点
- 引擎版本兼容性：新引擎版本升级后（如 Unity 2022 升级到 2023）功能是否正常、旧项目迁移到新引擎的适配问题
- 插件适配 bug：第三方插件（如广告插件、支付插件、analytics 插件）与引擎版本冲突、插件功能失效（如广告无法加载）
- 引擎自带功能异常：如 Unity Animator 动画状态机错误、Unreal 蓝图逻辑异常、引擎物理系统冲突

## 图形渲染技术

### 核心渲染技术
- 渲染管线：前向渲染（适合移动端，性能消耗低）、延迟渲染（适合主机 / PC，支持多光源）
- 特效技术：粒子特效（如火焰 / 爆炸）、体积雾 / 体积光、动态阴影（实时阴影 / 烘焙阴影）

### 测试点
- 光影 bug：光源照射异常（如物体无阴影）、动态光影闪烁 / 卡顿、不同场景光影过渡不自然
- 材质错误：材质显示异常（如透明 / 错位 / 颜色偏差）、材质贴图拉伸 / 模糊、材质切换（如角色换装）错误
- 分辨率适配：不同分辨率下 UI / 模型拉伸变形、高分辨率（如 4K）下渲染卡顿、分辨率切换时画面闪烁

## 网络通信技术

### 主流通信方式
- 传输协议：TCP（可靠传输，用于登录 / 支付等关键数据）、UDP（快速传输，用于联机对战实时数据）
- 网络架构：客户端 / 服务器（C/S）架构（如 MMORPG）、peer-to-peer（P2P）架构（如格斗游戏联机）

### 测试点
- 联机同步延迟：多玩家操作同步延迟（如 FPS 射击命中延迟）、角色位置同步偏差、技能效果同步滞后
- 数据包丢失：弱网环境下数据包丢失导致的功能异常（如角色卡住）、丢包后数据恢复逻辑（如重传机制）
- 断线重连：断网后重连成功率（目标≥95%）、重连后数据一致性（如进度不丢失）、重连超时处理

## 物理引擎技术

### 主流物理引擎
- Box2D：多用于 2D 游戏（如休闲益智、2D 动作），处理 2D 刚体碰撞、关节约束
- PhysX：多用于 3D 游戏（如 ACT、FPS），支持 3D 刚体运动、布料 / 流体模拟，Unreal 默认集成
- Bullet：开源物理引擎，可定制化，多用于自研引擎项目

### 测试点
- 碰撞检测错误：角色与场景物体碰撞失效（如穿墙）、碰撞体积与模型不匹配（如模型外碰撞）、快速移动物体碰撞漏检
- 刚体运动 bug：刚体运动速度异常（如漂浮 / 下坠过快）、重力模拟错误（如无重力 / 重力过大）、物体碰撞后穿透（如子弹穿过敌人）
- 穿模：角色与角色 / 场景穿模（如角色陷入地面）、装备 / 道具与角色穿模、大型物体（如车辆）穿模

## 数据存储技术

### 存储方式
- 本地存储：移动平台（SharedPreferences、PlayerPrefs）、PC / 主机（本地文件、注册表），用于保存玩家配置、本地存档
- 云端存储：云服务器数据库（如 MySQL、MongoDB），用于保存玩家账号数据、跨设备存档同步

### 测试点
- 存档丢失：正常退出 / 异常退出（如闪退）后存档是否丢失、清理缓存后存档是否保留、多账号切换存档混淆
- 数据读取错误：存档读取后数据错乱（如角色等级 / 道具异常）、读取速度过慢（如大型存档加载超时）、数据格式错误（如版本更新后旧存档无法读取）
- 云端同步 bug：跨设备同步存档失败、云端存档与本地存档冲突（如同步覆盖错误）、网络异常时云端同步中断后数据一致性

## 人工智能技术（AI）

### 游戏 AI 应用场景
- NPC 行为：友好 NPC（如商店老板、任务指引）、敌对 NPC（如怪物、敌人 AI）
- 决策逻辑：AI 战斗策略（如近战 / 远程切换）、路径寻路（如 NPC 绕开障碍物）、难度自适应（如根据玩家水平调整 AI 强度）

### 测试点
- NPC 行为异常：NPC 卡住（无法移动）、NPC 重复无效动作（如原地转圈）、NPC 无视玩家（如敌对 NPC 不攻击）
- AI 决策错误：AI 战斗逻辑混乱（如不攻击 / 盲目攻击）、路径寻路错误（如绕远路 / 无法到达目标点）、AI 协作异常（如队友 AI 不支援）
- 难度失衡：AI 难度过高（如新手玩家无法击败）、AI 难度过低（如后期玩家轻松获胜）、难度切换（如普通→困难）后 AI 强度无变化
